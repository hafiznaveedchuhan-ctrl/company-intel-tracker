# Company Intel Tracker

Real-time company news headline surveillance system powered by Tavily AI.

## Features
- Search any company for live headlines
- Auto-refresh at configurable intervals (30s to 1hr)
- AI-generated intelligence summary
- Relevance score badges (HIGH/MED/LOW)
- Source attribution with domain display

## Setup

1. Get a Tavily API key from https://tavily.com
2. Add to Vercel environment variables: `TAVILY_API_KEY=tvly-xxx...`

## Stack
- Next.js 14 (App Router)
- TypeScript + Tailwind CSS
- Tavily Search API
- Vercel deployment
